# Лабораторная работа №3: Двухсервисное веб-приложение (Онлайн-школа)

## Описание проекта
Данный проект представляет собой веб-приложение для управления учебными программами онлайн-школы. Приложение построено на базе клиент-серверной архитектуры и состоит из двух независимых микросервисов:
* **Backend (FastAPI)**: REST API для управления данными, строгая валидация входящих запросов с помощью Pydantic и взаимодействие с базой данных SQLite.
* **Frontend (Flask)**: Пользовательский веб-интерфейс (HTML/CSS), взаимодействующий с backend-сервисом по протоколу HTTP через библиотеку `requests`. Реализовано без использования JavaScript.

## Стек технологий
* **Язык программирования:** Python 3
* **Backend API:** FastAPI, Uvicorn, Pydantic
* **Frontend UI:** Flask, HTML5, CSS3, Requests
* **База данных:** SQLite3

# Установка и запуск программы

### 1. Склонируйте репозиторий и перейдите в папку проекта (для удобства за основу были взяты git команды)
`git clone https://github.com/ivasipersi-alt/learn_project`
`cd learn_project`

### 2. Создайте и активируйте виртуальное окружение:
`python -m venv .venv`
`.venv\Scripts\activate`

### 3. Установите библиотеки (зависимости) из файлика
`pip install -r requirements.txt`

### 4. Запускк Backend сервиса
Так как мы работаем в пайчарме, здесь есть 2 выбора запуска сервиса. Первый - с помощью интерфеса программы:
* создание двух конфигураций, где нужно указать пути к папке с main и app, а также в бекенде путь к директории
* с помощью команды:
`uvicorn backend.main:app --reload --port 8000`
### 5. Запуск Frontend сервиса 
Для запуска второго сервиса, необходимо создать второй терминал, где нужно активировать окружение (если нужно) и выполнить команду:
`python frontend/app.py`
* Интерфейс будет доступен по адресу :5000

### 6. Реализованный функционал
* Create: Добавление нового учебного курса (POST)
* Read: Просмотр списка всех курсов (GET)
* Update: Редактирование данных существующего курса (PUT/PATCH)
* Delete: Удаление курса из базы данных (DELETE)
