Лабораторная работа №2
Тема практики:
Разделение приложения на frontend и backend-сервисы с использованием APIFlask и Flask

Описание
    Проект состоит из двух сервисов:
         -backend представляет API и документацию в OpenAPI
        -frontend обращается к backend и отображает полученные данные в браузере
